Деплой на Хостинг Україна — чекліст
===================================

Перед збіркою
  [ ] Оновлено абсолютні URL у index.html (canonical, og:url, og:image, JSON-LD), якщо домен не cloud-pro.com.ua
  [ ] npm install виконано в корені проєкту

Збірка
  [ ] npm run build без помилок
  [ ] Перевірено наявність папки dist/ з index.html та assets/

Перед завантаженням
  [ ] Резервна копія поточного public_html (якщо там уже був сайт)
  [ ] Відомо точне ім’я кореневого каталогу сайту (public_html / www / інше)

Завантаження
  [ ] У каталог сайту завантажено ВМІСТ dist/ (index.html у корені каталогу)
  [ ] Права на файли дозволяють читання веб-сервером (типово 644 для файлів)

Після публікації
  [ ] Сайт відкривається по HTTPS
  [ ] Немає «голого» HTML без стилів (перевірка DevTools → Network)
  [ ] /robots.txt та /sitemap.xml відкриваються
  [ ] За потреби — перевірка в Google Search Console / аналогах
